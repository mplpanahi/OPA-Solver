OPA Solver Version 1.4: Multiple-Criteria Decision Analysis Software

Notes:

- Virtual Tour of the Software:

- This solver can be used for individual and group decision-making problems.

- If you only need to calculate the weight of criteria (the problem has not any alternative), please watch this video:

- This version cannot support the degree of preference. Please use continuous ranks (i.e., you cannot use 1, 2, 3, 5 because the ranks are not continuous)

- The weights of the experts and criteria show their role on the final decision (objective + subjective).

- Don't enter zero value as input data into the software.
